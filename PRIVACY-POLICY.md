# Chrome Tab Rotate - Privacy Policy

This Chrome extension collects limited, anonymous usage analytics to help improve the user experience.

## Data Collected

The extension collects standard Google Analytics usage data and a small number of custom events, including:
- When the extension icon is clicked
- When the extension cycles to the next tab

The data collected is anonymous and does not include:
- Personally identifiable information
- Browsing history
- Page content
- URLs of visited websites

## Purpose of Data Collection

Analytics data is collected solely to understand how the extension is used and to guide future improvements to functionality and usability.

## Data Storage and Sharing

All analytics data is collected and stored by Google Analytics.  
The data is not sold, shared, or transferred to any third parties.

## Third-Party Services

This extension uses Google Analytics for usage measurement.  
For more information about how Google handles data, please see:
https://policies.google.com/privacy

## Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be published at this location.
